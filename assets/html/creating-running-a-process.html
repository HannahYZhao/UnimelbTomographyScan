How do I create and run a Process?

At it's core, creating a Process is done by connecting multiple compatible Nodes together. A walkthrough of this process can be found in the Welcome message, but let's go through the details of how this all works

When you right click in the space where a Process can be created, a window will pop up listing all of the Nodes you can build your Process out of. These Nodes have been displayed in alphabetical order, and can be searched using your keyboard to enter text in the bar at the top of the window. When searching, it will try and find a Node who's name contains the text you wrote.

When you select a Node from this menu, it will then be placed inside the workspace. If you think the Node has been placed in the wrong spot, you can move it to a new location just by clicking and draging the node itself. Make sure not to click and drag on the ports though, as that is how you make connections with compatible nodes. Actually, why don't we go into that now.

To make a connection, click and drag from the port you want to connect to the port of the node you want to connect it to. If the node you want to connect to does not exist, trying to do so will prompt you to create a new Node to connect to instead. However, you can only create connections between ports that sent compatible data between them. For example, you cannot connect a port that outputs a .vtk file to a port that expects a folder directory. If you do try to do so though, an error message will pop up to let you know the types of the ports you were trying to connect.

Once a Process has been created, you can go to the Run menu at the top of the window and select the Run Process button. This will then run the process you have created.
