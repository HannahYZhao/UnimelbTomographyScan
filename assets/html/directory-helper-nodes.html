Directory Helper Nodes

Connector Node

How to use this node?

The job of this node is simple. As its name suggests, it passes a directory path without any modification. It can help maintain the readability of a process by avoiding long and staggered connection lines.


Copy File Node

How to use this node?

The function of this node is also intuitive, it receives a list of files and a directory path from its inputs. It will copy the files into the directory folder.


Create Folder Structure Node

How to use this node?

As one process may need to have several folders to save different kinds of files, this node will create a root folder as a workspace to manage files. It will receive a directory path from a Select Folder node, and it will create the following subfolders:

  • ini: to store the .ini file with the parameters chosen on the SPHARM node
  • low res vtk: to store the files generated by SlicerSalt when creating the low-res mesh
  • low res clean vtk: to store the low-res .vtk files
  • high res vtk: to store the files generated by SlicerSalt when creating the high-res mesh
  • high res clean vtk: to store the high-res .vtk files
  • TFMAT: to store the TFMAT files created in Calculate TFMAT node in Alignment Node
  • aligned: to store the aligned high-res .vtk files created by Aligned VTK Files node in Alignment Node
  • log: to store the log files

This node has eight outputs for the subfolders above respectively. Hence the user can connect them with other nodes to use these folders to store files.
