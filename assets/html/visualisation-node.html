Visualisation Node

The Visualisation VTK Image node allows a user to view and interact with .vtk images.

Loading Files

  • Except add files by Select VTK Files Node, you can also add files by click "Add File(s)..." button, This will open a window that will allow a person to select .vtk files. Once this has been done, the file name should appear on the left hand side of the window.
  • If you do not want the file names to appear on the left hand side, you can use the Remove Selected button on the left hand side.

Viewing Files

Once the given files are loaded, the corresponding file name should appear on the left hand side of the window with a checkbox. If the Visualise button is then selected, it will then render the selected images on the right hand side of the screen. The number of images to display per page can be configured with the Number of models to show on each pagedropdown.

  • If the dropdown is set to display 1 image per page, then it will show the image in 4orientations: its original orientation as well as sagittal, transversal, and coronal.
  • If more than one image is shown per page, then only 1 orientation is shown. Users can choose the orientation using the dropdown at the top next to the camera button.
  • Once the dropdown is set, and the rendered file would show up on the right hand side.

Interacting with Images

  • Camera button could reset all images to their original orientation
  • There are 3 buttons for flipping about the x, y, and z axes.
  • Axes button could customize axes color and the transparency of the bone.

Generating Log File

  • Once rendered, each image will have a Log button. When selected, a File Icon should appear on the left hand side near the file name.
  • Once all desired images have been selected, you can use Create Log File node to generate a log file with a timestamp and the list of all file names selected to be logged.
